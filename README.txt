Repository to share basic Matlab functions with the team :

1- ReadTiff.m: Read acquisition.tif file with several images in the same .tif file

2- ReadManyTif.m: Read all .tif files of a folder and store them in a single 3-D matrix 

3- SaveTif.m: Save a 2-D or 3-D matrix to a single .tif file

4- SetTag.m : General parameters to save the .tif files (Mandatory for SaveTif.m)
